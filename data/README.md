# Dataset

## Description
This dataset contains 5 years of daily stock market data for multiple companies,
including Open, High, Low, Close prices and Volume.

## Features
- `Date` - Trading date
- `Open` - Opening price
- `High` - Day's highest price
- `Low` - Day's lowest price
- `Close` - Closing price (target variable is next day's close)
- `Volume` - Number of shares traded
- `Name` - Company ticker/name
